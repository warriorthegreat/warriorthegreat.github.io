# JAVA筆記2 (變數~)

#### 034 變數原理
* 變數是程式的基本 : 
    * 具有資料型態、名字、數值。
    * 例如 int a = 1 的意思是說，在記憶體裡面有一個位置會被定義為變數a，而這個位置將只能儲存int(整數)，而存進去的數字是1。

#### 035 變數概念
* 變數就是記憶體儲存的表示方法。
* 變數就像門牌，透過找到門牌可以找到房子裡的居民(數值)。
* 用變數的步驟 : 
    * 宣告變數以及資料型態 (如 資料型態為int，變數名字為a。)
    * 給變數數值。(int a = 5，給 變數a數值5)
#### 036 變數快速入門
* 變數的型態 : 
    * int 整數 
    * double 雙倍精準數
    * float 單倍精準數(如果有記憶體限制用這個比較省空間)
    * char 字元
    * string 字串

#### 037 變數注意事項
* 變數表示記憶體內的儲存空間。
* int 用4bytes，double 用8bytes。
* 變數必須要先宣告才能使用(宣告名稱以及資料類型)
* 變數在一個作用域(Scope)內不能重複命名。
